## ✨ Features

* **Interactive Particle Background:** A custom-built, lightweight HTML5 Canvas particle system that connects nodes dynamically.
* **Gamified Certification Deck:** A unique, interactive 3D card-dealing UI to display professional certificates with draw, flip, and burst confetti animations.
* **Typing Animation Hero:** Engaging "Hello World" drop-in loader followed by a dynamic text-typing effect in the hero section.
* **Neon Aesthetic UI:** Custom CSS variables driving a sleek dark mode theme with glowing cyan, purple, and pink accents.
* **Direct Resume Download:** Integrated access for recruiters to download my resume instantly.
* **Fully Responsive:** Optimized for both desktop and mobile viewing with a mobile-friendly navigation toggle.

## 🛠️ Tech Stack

* **Frontend:** HTML5, CSS3 (Flexbox, Grid, Custom Properties, Keyframes), Vanilla JavaScript (ES6+).
* **Styling & Icons:** FontAwesome 6.4.0, Google Fonts (Cinzel Decorative, Rajdhani, Space Mono).
* **Architecture:** Zero heavy frameworks; pure DOM manipulation and `requestAnimationFrame` for high-performance rendering.
